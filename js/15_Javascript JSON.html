# JSON

  • Javascript Object Notation, 자바스크립트식 객체 표기법
  • 언어에 의존하지 않고 다른 플랫폼과 손쉽게 데이터를 주고받을 수 있다.
  • Javascript Object처럼 생겼지만 텍스트만 사용한다. (언어 독립적이어야 하기 때문에, 각 언어의 문법은 사용할 수 없다.)

# Javascript Object vs JSON

- 둘 모두에서 가능한 것들
  • string
  • number
  • object
  • array
  • boolean
  • null
  • "" (double quatation)
- Javascript에서만 가능한 것들
  • function
  • undefined
  • '' (single quatation)
  • ...등 다른 언어에는 없는 것들
