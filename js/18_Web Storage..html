# Cookie

  • 사실 쿠키의 본래 용도는 브라우저에 저장만 해두고, 서버로 데이터를 전송할 때 포함시키는 용도이다.
  • 하지만 자바스크립트로도 제어할 수 있다.
  • 다음에 하도록 하자
    • ** 크롬에서는 도메인이나 https 통신이 없는 경우 쿠키 저장을 하지 않는다. 즉, 로컬 파일로 열린 코드(file://)에서는 쿠키 저장이 안됨

# localStorage

  • 브라우저에 데이터를 보관하는 용도
  • 서버로 일일이 전송할 필요없는 정보를 저장하는 것이 좋다.
  • sessionStorage는 브라우저가 닫히면 데이터가 사라진다는 점만 다르다.
