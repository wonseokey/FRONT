CDN (Content Delivery Network)

  • 컨텐츠를 네트워크를 통해 여러 사용자에게 제공하는 것
  • 컨텐츠 제공 회사가 제공하는 링크로 글꼴, 아이콘, 이미지 등을 가져다 사용할 수 있다
  • 모든 컨텐츠를 내 서버에 저장하는 것보다 더 효율적이다

# 구글 아이콘 (material.io)

  • 아이콘은 실제로는 글꼴이기 때문에 글자에 대한 스타일이 모두 적용된다
  • family=Material+Icons
  • family=Material+Icons+Outlined
  • family=Material+Icons+Two+Tone
  • family=Material+Icons+Round
  • family=Material+Icons+Sharp
  • (여러개 적용시) family=Material+Icons|Material+Icons+Outlined|Material+Icons+Two+Tone|Material+Icons+Round|Material+Icons+Sharp
